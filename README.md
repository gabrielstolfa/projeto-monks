# Projeto Monks

Projeto de uma landing page responsiva desenvolvido como desafio técnico, com foco em estruturação semântica, organização visual, responsividade e interações em JavaScript.

## 🔗 Acesse o projeto

- Deploy: [Projeto Monks](https://gabrielstolfa.github.io/projeto-monks/#formSection)
- Repositório: [gabrielstolfa/projeto-monks](https://github.com/gabrielstolfa/projeto-monks)

## 📖 Sobre o projeto

O **Projeto Monks** é uma página institucional/promocional desenvolvida para simular uma experiência moderna de apresentação de produto ou serviço.

A aplicação foi construída com foco em:

- Estruturação de layout em múltiplas seções
- Design responsivo
- Organização visual de conteúdo
- Navegação clara
- Interações básicas com JavaScript
- Boas práticas com HTML, CSS e JS puro

## ✨ Funcionalidades

- Menu de navegação
- Seção principal de destaque (hero)
- Cards de produtos/categorias
- Bloco visual com imagens em destaque
- Links para lojas de aplicativos
- Lista de categorias
- Seções informativas
- Área de formulário
- Verificação de segurança no formulário
- Rodapé com redes sociais

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub
- GitHub Pages
